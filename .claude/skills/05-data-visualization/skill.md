# Skill: Data Visualization & Dashboards

## Goal
Build rich data dashboards with charts, tables, KPI cards.

## Stack
- Recharts for charts
- shadcn Table for data tables
- StatCard component for KPI metrics

## Patterns
- Grid of StatCards with stagger animation
- Generic Table<T> with column definitions
- Theme-aware chart colors from CSS variables

## DoD
- Charts render correctly in dark/light theme
- Data tables handle empty states
- Responsive on all screen sizes
